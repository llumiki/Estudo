Classes

Na verdade no Skyrim não existem classes realmente, você livre para o que quer vestir ou usar como arma, mas existem classes de habilidades, sendo elas as habilidades de guerreiro, de mago e de ladrão, para cada classe de habilidade existem seis árvores de habilidades, cada árvore tem um nível individual que é evoluído ao realizar a ação equivalente a árvore, por exemplo, se eu receber dano usando uma armadura de ferro, eu vou ganhar experiência na árvore armadura pesada. Além das habilidades o jogo conta também com equipamentos, tendo espaços para: elmo; peitoral; manoplas; botas; anel; colar. Tendo também um espaço para cada mão, podendo usar ou uma espada e uma magia, ou cajado e um escudo, ou duas adagas, você é livre para montar sua classe do jeito que quiser.
